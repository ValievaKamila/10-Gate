# 10-Gate
ЗАДАНИЕ: Реализовать веб-страницу с воротами. Изначально на странице отображаются два одинаковых блока, за которыми спрятан текст. При наведении мышкой на любой из блоков они медленно (за 2 секунды) раскрываются так, чтобы был виден текст. Если мышку увести, то всё плавно возвращается к первоначальному состоянию. ТЕМА: Плавные переходы. Анимации. Трансформации. Фильтры.
